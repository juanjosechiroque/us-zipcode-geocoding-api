# Dataset

`data/us_zip_codes.csv` is derived from the GeoNames US postal-code dataset.

- Source: https://download.geonames.org/export/zip/US.zip
- Download date: `2026-07-20`
- Records in the repository CSV: `41,488`
- License: Creative Commons Attribution 4.0 International —
  https://creativecommons.org/licenses/by/4.0/

The source was converted to CSV, restricted to the fields used by this API, and
deduplicated by `zip_code`. GeoNames does not endorse this project, and the data is
provided without warranties of accuracy, timeliness, or completeness.

When refreshing the dataset, update the download date and record count in this document.
